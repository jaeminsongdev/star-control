# Star-Control

Star-Control은 사용자가 Codex 앱에 입력한 개발 목표를 실행 단계로 나누고, 각 단계에 알맞은 모델·생각 깊이·실행 방식·검사 방법을 배정하는 Windows용 개발 제어 도구다.

이 프로젝트는 Codex를 대체하거나 새로운 AI 실행기를 만들지 않는다. Codex가 작업을 수행하고, Star-Control은 작업의 시작·분할·배정·안전·검사·기록·이어하기를 담당한다.

현재는 문서 설계 단계다. 새 설계의 기준은 [문서 안내](docs/README.md)에서 순서대로 확인한다.

## 현재 원칙

- Codex 전용이다.
- 로컬 AI와 다른 AI 제공자는 지원하지 않는다.
- OpenAI API를 직접 호출하지 않는다.
- Windows만 지원한다.
- 브라우저 화면은 만들지 않고 Codex 앱과 터미널을 사용한다.
- 레거시는 로컬 참고자료일 뿐 현재 설계 기준이 아니다.
- 코드 구현은 문서 설계가 확정된 뒤 시작한다.
