# 선별 재검토 체크리스트

기능 추가/삭제 판단 시 아래 질문을 적용한다.

- [ ] AI 주 구현자 체계에서 실제 사고를 줄이는가?
- [ ] 1인 최종검토 시간을 줄이는가?
- [ ] 여러 프로젝트에 재사용 가능한가?
- [ ] 오탐 관리를 할 수 있는가?
- [ ] 구현 비용이 장기 효용보다 작거나 정당화되는가?
- [ ] 조건부 기능이라면 트리거가 명확한가?
- [ ] 외부 표준/자료에서 구현 힌트를 얻었는가?
- [ ] P0/P1과 독립적으로 먼저 구현해야 할 이유가 있는가?
