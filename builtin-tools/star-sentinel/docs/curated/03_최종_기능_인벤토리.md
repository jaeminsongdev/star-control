# 03. 최종 기능 인벤토리

v4 기준 최종 장기 구현 대상은 **75개 기능**이다.

- P0: 즉시 기반/핵심 안전장치
- P1: P0 이후 바로 구현할 일반 품질/계약/보안 기반
- P2: 프로젝트가 커지면 구현할 신뢰성/공급망/운영 기능
- P3: 특정 조건에서만 구현할 고급 기능

| ID | 기능 | 분류 | 단계 | 필요한 이유 | 구현 요약 | 트리거 |
|---|---|---|---|---|---|---|
| F001 | 저장소 루트/프로젝트 인식기 | 기반/저장소 | P0 | 모든 검증은 프로젝트 구조를 알아야 작동한다. | root marker(.git, AGENTS.md, Cargo.toml/package.json 등) 탐지, workspace 목록, 언어/빌드 시스템 추정, 결과를 repo_map.json으로 저장. | 항상 |
